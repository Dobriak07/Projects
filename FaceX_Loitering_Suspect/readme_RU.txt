1. Скопировать папку "node_modules" по пути "C:\Program Files (x86)\ISS\SecurOS\node.js"
2. В SecurOS создать объект NodeJS Script
3. Скопировать содержимое скрипта "main_v2.js" в скрипт в SecurOS
4. IP и порты получаем из объектов "Компьютер" и "FaceX Server" . DNS-имя компьютера может вызвать проблему (не резолвиться корректно), для исправления - в поле IP свойств объекта "Компьютер" прописать или подставить корректный IP. Перезапустить скрипт.

Другие переменные:
 "SUSPECT_LIST" - имя списка, в который помещаются подозреваемые
 "SCAMMERS_LIST" - имя списка, в который скопируются подозреваемые после достижения "detectionCount"
 "matchThreshold" - процент совпадения захвата с контрольным списком. Для детекции персон в масках рекомендуется значение 0.5
 "monitoringTime" - время мониторинга персоны из списка подозреваемых, по истечении времени персона будет удалена из нее, если счетчик детекций не сравнялся или превысил "detectionCount"
 "watchdog" - как часто запускаем вспомогательную функцию по проверке достижения "monitoringTime". По-умолчанию 10с, ниже ставить не стоит.
 "detectionCount" - сколько раз подозреваемый должен быть задетектирован для переноса в "SCAMMERS_LIST"

Путь сохранения логов скрипта - "C:\ProgramData\ISS\logs\modules\Suspect". 
Уровень логирования, размер логов и их ротация настраиваются в разделе "Logger section"
