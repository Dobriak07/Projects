Краткая инструкция по разворачиванию интеграции VideoIntellect в SecurOS Astra Linux:
1. После распаковки архива запускаем терминал, для удобства переходим в папку, куда распокавали архив
2. Выполняем комманду "sudo chmod +x ./VI_copy.sh", чтобы баш-скрипт был выполняемым
3. Запускаем баш-скрипт "sudo ./VI_copy.sh". Баш-скрипт скопирует папку node_modules по пути /opt/iss/securos/node.js/ и даст права
4. Создаем в SecurOS в разделе "Интеграция и автоматизация" скрипт NodeJS
5. Вставляем содержимое файла из одного из вариантов скрипта: 
 - VI_integration_live_drawing.txt - Вариант с отрисовкой фигур на камерах живого видео
 - VI_integration.txt - Вариант без отрисовки, только события
6. Далее по комментариям в скрипте, достаточно указать host - адрес сервера, где установлен VideoIntellect.
P.S. Логи сохраняются по пути "/var/opt/securos/logs/VI"
